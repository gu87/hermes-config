# 剧本时间线与人物准确性准则

## 为什么需要这些规则

贝克汉姆 H5 测试篇暴露了两个问题：用户反馈图片"不匹配那几年的形象"（根因是 prompt 缺少年份锚点），同时剧本时间线也有硬伤（跳过三冠王赛季、弗格森电话时序模糊）。**人物类叙事游戏的准确性是用户体验的基石**。

## 时间线检查清单

### 1. 每个关键事件前标注年份

```diff
- "他陷入了深深的抑郁中..."
+ "1998年夏天。他陷入了深深的抑郁中..."
```

即使读者能推理出年代，也要显式标注。格式统一：`"yyyy年，[季节/月份]。"` + 事件描述。

### 2. 重大事件不能"一笔带过"

**反例**（踩坑重现）：
```javascript
// ❌ 只用一个 narrate 跳过 3 年
{ type: "narrate", text: "三年过去了。1999年贝克汉姆随曼联夺得欧冠冠军。" }
```

**正确做法**——按真实历史拆分：
```javascript
// ✅ 赛季表现 → 关键比赛 → 里程碑
{ type: "narrate", text: "新赛季开始了。每当贝克汉姆拿球，客场的球迷就报以漫天的嘘声..." }
{ type: "narrate", text: "1999年5月26日，巴塞罗那诺坎普球场。欧冠决赛..." }
{ type: "card", card_type: "event", title: "诺坎普奇迹·三冠王", ... }
```

**规则**：某段时间内 ≥2 个重大事件 → 至少用 2 个 narrate + 1 个 card/hero 来展现，不能浓缩到一句旁白。

### 3. 重大事件顺序不可调换

| 事件 | 真实时间 | 常见错误写法 |
|------|---------|------------|
| 弗格森电话安抚 | 1998年7月（世界杯后立即） | 放到低谷期数月之后 |
| 三冠王赛季 | 1998-99赛季 | 只提一句"三年过去了" |
| 国家队队长任命 | 2000年（三冠王之后） | 随意放置 |
| 希腊任意球救赎 | 2001年10月 | — |
| 阿根廷点球复仇 | 2002年6月 | — |

### 4. 对话/内心独白的时间锚点

每句独白需对应当时的真实历史时间点：

```
hero_6: "他们说我毁了英格兰。也许他们是对的。"
→ 发生在1998年7-8月，不是1999年

hero_8: "又是任意球。整整三年，我等的就是这个瞬间。"
→ 2001年10月，距1998年红牌确实3年多 ≈ 合理
```

## 人物命名

| 常见错拼 | 正确写法 | 备注 |
|---------|---------|------|
| Simoni / Simeoni | **Simeone** | Diego Pablo Simeone（阿根廷，1998世界杯） |
| C.罗 c罗 | **Cristiano Ronaldo** | 中文文中也用"C罗" |
| 贝克汉姆(中文) | **Beckham** | speaker 字段用英文原名 |

> **注意**：Simeone 的正确定义是 "Simeone"（迭戈·西蒙尼），不是 "Simoni" 或 "Simeoni"。1998 年世界杯与贝克汉姆冲突的就是他。剧本 speaker 字段用 `"Simeone"`。

**规则**：剧本中首次出现用全名，speaker 字段用英文原名（如 `"Ferguson"` 而非 `"弗格森"`），西班牙/意大利球员注意姓氏拼写。球员名字务必查 Wikipedia 确认正确姓氏。

## 事实核查流程

皮尔洛出完剧本后执行：

1. 提取所有年份/日期/比赛名
2. 逐条核对维基百科或懂球帝资料
3. 标注可能的时间线冲突
4. 对跳过的重要事件段补充分解节点

### 5. 比赛场地名必须精确（已踩坑）

**反例**：hero_1 的 prompt 写 "Stade de Marseille" → 实际 1998 年英格兰 vs 哥伦比亚在 **Stade Félix-Bollaert, Lens**

**规则**：每个比赛节点必须核查具体球场名，不在 prompt 或旁白中用城市名代替球场名。

### 6. 同一人物不同年份的视觉锚点分段

不要为所有年份写统一的 `"A portrait of XXX"` 模板。每个图片 prompt 的年份/发型/球衣/球场必须独立核查。

### 7. 预选赛 vs 正赛 — 比赛性质必须准确

**反例**（踩坑）：hero_8 prompt 写 "at 2002 FIFA World Cup, about to take a free kick against Greece"。
- 希腊 2002 年世界杯预选赛被英格兰淘汰，根本**未晋级正赛**
- 这场是 **2001 年 10 月 6 日的世界杯预选赛**（qualifier），不是世界杯正赛（tournament）
- 正赛在韩日，预选赛在老特拉福德

**规则**：每场比赛在写 prompt/旁白前，先确认是：
- 预选赛（qualifier）— 通常在参赛国主场
- 正赛（tournament / World Cup finals）— 在主办国
- 不要同时出现"2002 World Cup"和"Greece/Greek"——说明比赛性质混淆

### 8. 足球运动员发型时间线核查

**规则**：每个年份的发型必须独立核查，不能用该人物最出名的造型覆盖所有年代。

**贝克汉姆发型时间线（已验证）：**

| 年份 | 发型 | 球衣 | 赛事/场景 |
|------|------|------|----------|
| 1998 | 金色板寸（short bleached buzz cut） | 英格兰白#7 | 法国世界杯 |
| 1999 | 稍长金发偏分（longer blonde, side-parted） | 曼联红 | 赛季/三冠王/欧冠 |
| 2000 | 相似风格，发色更自然 | 英格兰白#7 / 曼联红 | 欧洲杯/赛季 |
| 2001 | 短刺金发（short spiky bleached） | 英格兰白#7 / 曼联红 | 希腊预选赛 |
| 2002 | 莫西干金发（iconic bleached mohawk） | 英格兰白#7 | 韩日世界杯 |
| 2003-04 | 偏分深金发（longer darker blonde, roots showing） | 皇马白 | 转会皇马初期 |
| 2004 | 短发鬓角剃（short with shaved sides） | 英格兰白#7 | 欧洲杯 |

**⚠️ 关键分界**：莫西干头是 **2002 世界杯专属**发型，2001 年没有。写 prompt 时不要跨年借用。
