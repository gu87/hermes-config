# Humanizer-zh — 中文 AI 文本润色

## 使用时机

皮尔洛出完方案/文案初稿后，交付前最后一步。或者任何 AI 生成的中文内容需要转成自然表达时。

## 润色原则

1. **去掉 AI 腔标记词**：删除「值得注意的是」「毋庸置疑」「综上所述」「在这个背景下」「从某种角度来说」等空洞过渡词
2. **缩短长句**：超过 40 字的句子拆成 2-3 句
3. **去重**：同一观点在段落中只出现一次
4. **口语化但不失专业**：把「我们建议采用以下策略」→「建议这么干」；把「通过…从而实现…」→「用…来…」
5. **保留数据**：所有数字、来源、日期、金额不变
6. **保持人称一致**：统一用「我们」或品牌名，不混用
7. **连接词精简**：去掉不必要的「而」「且」「则」「以」「其」

## 输出格式

输入一段 AI 中文文本，输出：
- 润色后的文本（只输正文，不加说明）
- 如果是长文（>500字），在开头加一句摘要

## 示例

**输入：**
```
值得注意的是，在当今数字化营销的大背景下，懂球帝通过整合多平台资源，从而实现了品牌影响力的显著提升。毋庸置疑，这一策略为我们在激烈的市场竞争中赢得了先机。
```

**输出：**
```
懂球帝通过整合多平台资源，提升了品牌影响力。这个策略帮我们在激烈的市场竞争中抢到了先机。
```

## 禁止行为

- 不改变原文事实和数据
- 不添加原文没有的信息
- 不改变专业术语和品牌名
- 不过度口语化到失格（如「绝绝子」「YYDS」）
